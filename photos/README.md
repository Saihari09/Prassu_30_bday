# Photos

Drop your photos in this folder. Then reference them in `content.js` like:

```js
photo: 'photos/first-date.jpg'
```

Tips:
- JPG or PNG, ideally under 500 KB each (use a tool like https://squoosh.app to compress)
- Roughly 4:3 or square aspect ratio looks best in the Firsts cards
- Filenames: lowercase, no spaces — `first-kiss.jpg` not `First Kiss.jpg`
